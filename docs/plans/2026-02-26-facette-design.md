# Facette - Design Document

## Overview

**Facette** is a .NET Roslyn source generator library that combines DTO generation and mapper code generation into a single tool. Inspired by [Tim-Maes/Facet](https://github.com/Tim-Maes/Facet) (DTO generation) and [Riok/Mapperly](https://github.com/riok/mapperly) (compile-time mapper generation), Facette is a new standalone library that cherry-picks the best ideas from each.

The name "Facette" is the French diminutive of "facet" - a nod to the gem/light metaphor where a single domain entity can be viewed through different facets (DTOs).

## Core Concept

You declare a partial record with a `[Facette]` attribute pointing at a source entity. The generator fills in:
1. **Properties** on the partial record (mirroring the source, minus exclusions)
2. **Inline mapping methods** (`FromSource`, `ToSource`) directly on the DTO
3. **A static projection expression** for LINQ/EF Core `IQueryable` usage
4. **A companion mapper class** with extension methods (`user.ToDto()`, `query.ProjectToDto()`)

Zero reflection, zero runtime overhead, fully AOT-compatible.

## User-Facing API

### Basic Usage

```csharp
// Domain entity
public class User
{
    public int Id { get; set; }
    public string FirstName { get; set; }
    public string LastName { get; set; }
    public string Email { get; set; }
    public string PasswordHash { get; set; }
    public DateTime CreatedAt { get; set; }
}

// Declare a DTO - generator fills in properties + mapping
[Facette(typeof(User), exclude: nameof(User.PasswordHash))]
public partial record UserDto;
```

### `[Facette]` Attribute (v1)

```csharp
[AttributeUsage(AttributeTargets.Class | AttributeTargets.Struct, AllowMultiple = false)]
public sealed class FacetteAttribute : Attribute
{
    public FacetteAttribute(Type sourceType, params string[] exclude) { }

    /// <summary>Whitelist of properties to include (mutually exclusive with exclude).</summary>
    public string[]? Include { get; set; }

    /// <summary>Whether to generate ToSource() reverse mapping. Default: true.</summary>
    public bool GenerateToSource { get; set; } = true;

    /// <summary>Whether to generate the static Projection expression. Default: true.</summary>
    public bool GenerateProjection { get; set; } = true;

    /// <summary>Whether to generate the companion mapper class. Default: true.</summary>
    public bool GenerateMapper { get; set; } = true;
}
```

## Generated Code Shape

### Input

```csharp
[Facette(typeof(User), exclude: nameof(User.PasswordHash))]
public partial record UserDto;
```

### Output File 1: `UserDto.g.cs`

```csharp
// <auto-generated />
#nullable enable

namespace MyApp.Dtos;

public partial record UserDto
{
    public int Id { get; init; }
    public string FirstName { get; init; } = default!;
    public string LastName { get; init; } = default!;
    public string Email { get; init; } = default!;
    public DateTime CreatedAt { get; init; }

    public static UserDto FromSource(User source)
    {
        return new UserDto
        {
            Id = source.Id,
            FirstName = source.FirstName,
            LastName = source.LastName,
            Email = source.Email,
            CreatedAt = source.CreatedAt
        };
    }

    public User ToSource()
    {
        return new User
        {
            Id = this.Id,
            FirstName = this.FirstName,
            LastName = this.LastName,
            Email = this.Email,
            CreatedAt = this.CreatedAt
        };
    }

    public static System.Linq.Expressions.Expression<System.Func<User, UserDto>> Projection =>
        source => new UserDto
        {
            Id = source.Id,
            FirstName = source.FirstName,
            LastName = source.LastName,
            Email = source.Email,
            CreatedAt = source.CreatedAt
        };
}
```

### Output File 2: `UserDtoMapper.g.cs`

```csharp
// <auto-generated />
#nullable enable

namespace MyApp.Dtos;

public static partial class UserDtoMapper
{
    public static UserDto ToDto(this User source)
        => UserDto.FromSource(source);

    public static User ToSource(this UserDto dto)
        => dto.ToSource();

    public static IQueryable<UserDto> ProjectToDto(this IQueryable<User> query)
        => query.Select(UserDto.Projection);
}
```

### Design Decisions

- **`init` properties** on records for immutability by default
- **`= default!`** for reference types to suppress nullability warnings
- **Plain property-by-property assignment** (no reflection, like Mapperly)
- **Extension methods** on the mapper class for ergonomic `user.ToDto()` syntax
- **Projection delegates** to the static expression on the DTO (single source of truth)

## Architecture

### Package Split

| Package | Purpose | Targets |
|---------|---------|---------|
| **Facette.Abstractions** | Attributes only (tiny, no dependencies) | `netstandard2.0;net8.0;net9.0` |
| **Facette** | Source generator + analyzers | `netstandard2.0` (packed as `analyzers/dotnet/cs`) |

### Generator Pipeline

1. `ForAttributeWithMetadataName("Facette.Abstractions.FacetteAttribute")` - discover annotated types
2. `ModelBuilder.Build()` - parse attributes into `FacetteTargetModel`
3. Emit two source files per target:
   - `{TypeName}.g.cs` - partial record with properties + inline methods
   - `{TypeName}Mapper.g.cs` - companion mapper class

## Diagnostics (v1)

| Code | Severity | Description |
|------|----------|-------------|
| `FCT001` | Error | Type must be declared as partial |
| `FCT002` | Error | Include and Exclude cannot both be specified |
| `FCT003` | Error | Source type not found or inaccessible |
| `FCT004` | Warning | Excluded/included property not found on source type |

## Testing Strategy

- **Snapshot tests**: Feed source code into the generator, verify generated output matches expected snapshots (using `Microsoft.CodeAnalysis.CSharp.Testing` + Verify)
- **Integration tests**: Compile generated code and execute mappings at runtime
- **Analyzer tests**: Verify diagnostics for invalid usage

## Scope (v1)

### In scope
- Property generation (include/exclude)
- Basic name-matched mapping (property names must match)
- `FromSource()` / `ToSource()` inline methods
- Static `Projection` expression for LINQ
- Companion mapper class with extension methods
- `IQueryable<T>` projection via mapper
- Diagnostics for common errors
- Partial record as default output type

### Out of scope (future versions)
- Nested object mapping (Address -> AddressDto)
- Collection mapping (List<T> -> List<TDto>)
- Enum conversion
- Flattening / unflattening
- `[MapFrom]` / `[MapProperty]` custom property mapping
- Nullable properties mode
- Copy attributes from source
- Wrappers / CRUD DTO sets
- Expression mapping
- EF Core integration package
- Conditional mapping
- Before/after map hooks
